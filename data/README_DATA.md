# Data Package

This zip contains anonymized solar plant operations and financial data for two companies.

Each company folder contains:

- suggested users with different access scopes
- `api/` with raw API-style JSON exports for plants, elements, datasources, and time-series data
- `financial/` with hourly market prices and monthly costs

Energy-only users should not be able to access financial data such as market prices, revenue, or monthly costs.
